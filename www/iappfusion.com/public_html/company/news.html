6/7/2011 Louisville, KY

Louisville, Kentucky's local mobile app company, iAppFusion, LLC, is proud to announce Pondwater™ 2.0.

Pondwater is iAppFusion's proprietary, secure database storage system stored "In The Cloud". The Pondwater platform originated for gathering sensor data over four years ago, and has undergone extensive changes in the last year to adopt it for mobile data and mobile apps. It allows iAppFusion to store data from multiple sources and users and provides a secure transaction layer to access the data. The latest version of Pondwater was specifically designed to ease the pain of integrating disparate databases across multiple, mobile platforms and provide robust reporting and data analytics. With this tool, we are able to merge and store different, large-scale data sets with ease.

It has allowed iAppFusion to create "Framework" apps, into which different data sets can be dropped in. For example, the company created a beer database app called BrewHaHa. iAppFusion was then able to take the framework and drop in a cigar database for The Humidor app. iAppFusion can take any database, do some re-skinning on the app and publish. It takes roughly two weeks per app. This has been deployed on the iPhone/iPod/iPad and Windows Phone 7 platforms.

Additionally, it also serves as our proprietary set of API's that enables secure data access and communication. Each app or project or app has it's own set of API's. Some examples of API’s for current projects can be viewed at www.duckstick.com.

When an app is first used, it hits Pondwater via a Web Service. An account is then created for access to the data, a unique GUID for each mobile device is stored and a unique key is issued for each device. Based on the person's credentials, iAppFusion's back-end system manages and maintains exactly what data each user is limited to. As more data permissions are granted to an individual, Pondwater, in turn, upgrades the license to afford the user additional data access.

One of the biggest benefits to using Pondwater is it gives iAppFusion the ability to collect real-time consumer patterns. Each time an app is used, iAppFusion collects the location, the amount of time the app was used, a time/date stamp and the features used. This data can be stored and used for reporting and data analytics to provide very powerful marketing information.

Pondwater can be licensed to third parties to run on their own servers. Contact iAppFusion for details on the exciting database innovation that is Pondwater.

1/10/2011 Louisville, KY

Louisville, Kentucky's local mobile app company, iAppFusion, LLC, is proud to announce its partnership with Critical Technologies Inc., a technology firm based in Utica, NY.

iAppFusion and CTI will be integrating key components of each company's technological advances to gain economies of scale and produce even faster, cutting-edge, mobile technology.

About iAppFusion: iAppFusion, LLC is a mobile app firm, created by 3 local entrepreneurs and is dedicated to putting the power of the cloud into the palm of everyone's hands. This new company creates international mobile apps across multiple languages and platforms: iPhone, iPad, iPod, Android, Blackberry and Windows Phone 7. iAppFusion is an expert and innovator in mobile location services products. Go to www.iAppFusion.com to learn more.

Critical Technologies, Inc (CTI) is a veteran Central New York firm, focusing on the convergence of experience in sensor, storage, and wireless networking in order to deliver "Transparent Reliability" for wireless network users. CTI has developed and deployed solutions which enable the preservation and acceleration of user access to network data and services for military mobile and ad hoc networked platforms. CTI's Disruption / Delay / Disconnection Tolerant Networking (DTN) and Concurrent Multipath Routing (CMR) software manage the interrupted data flows caused by the variable, intermittent, asymmetric, and degraded wireless links encountered in the real world, delivering automated reliability.
